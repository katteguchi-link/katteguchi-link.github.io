What is katteguchi.link?

Katteguchi (勝手口) means "back door" in Japanese — the informal, practical entrance that gets you where you need to go without the fuss.

🌐

x402 API Access

We provide x402-powered APIs so your AI agents can pay-per-request to access Japan-specific data sources — instantly, programmatically, and without friction.

🗾

No VPN Needed

Many Japanese services are geo-restricted. Instead of wrestling with VPN connections, let katteguchi.link handle the Japan-side access for you.

🤖

Built for AI Agents

Designed from the ground up for autonomous AI agents. Simple REST endpoints, pay-per-request via x402 — no API keys, no sign-ups, no OAuth flows.

How It Works

1

Call the API

Your AI agent sends an HTTP request to a katteguchi.link endpoint.

2

Pay via x402

The request includes an x402 payment header. No accounts, no API keys — just crypto-native micropayments.

3

Get Japan Data

katteguchi.link fetches the data from Japan-restricted sources and returns it to your agent.
